About the projects

Throughout my training as a web developer, I’ve worked on different projects that have helped me gain experience with various technologies. In this portfolio, I’m going to show three of them, explaining a bit about what they are, which technologies I used, and how they were carried out. Each project helped me face different challenges and learn new ways to solve problems, which has been key to my professional growth.

Projects

StockMaster

My latest project is a product management web application where users can have multiple warehouses with their respective products. Based on the stored data, charts are generated automatically, and thanks to the integration of artificial intelligence, it is possible to create any kind of custom chart.

Additionally, it features an AI-powered chat system that allows users to ask questions about how the site works or to inquire about product-related information, such as available stock, the most expensive product, and more.

One of the main features is the scanner, which enables users to add or remove products from warehouses simply by scanning their barcodes.

Alicia Estetica

This was a group project consisting of a website for an aesthetic clinic. One of its main features is the product store, where administrators can add new products.

It also includes a booking form that only allows selecting available days and times. Once another user has booked an appointment with a professional at a specific date and time, that slot becomes unavailable.

Finally, the homepage displays all the services offered by the clinic, organized clearly and accessibly for users.

EcoBuddy

This project was developed as a team. It is a social network aimed at encouraging people to take positive actions for the environment.

Users can join activities such as trash collection, tree planting, and many other community-driven initiatives. It also features a rewards system that motivates continued participation in these activities.

The ultimate goal of this social network is to promote these kinds of actions and raise awareness about the importance of taking care of the environment.

StockMaster -- Inventory Manager

This was my final project for DAW, to which I dedicated a lot of time and from which I learned a great deal. I used multiple libraries and APIs, and deepened my knowledge in technologies I was already familiar with, such as Angular and Symfony.

A standout feature of this project is the use of AI. Thanks to this, I developed an intelligent notification panel that alerts the user to any issues—from simple errors like a misspelled name in the database to more critical problems like low stock or inconsistent prices. Additionally, I implemented a chatbot with access to all user data, capable of answering questions related to the products, as well as an AI-powered chart generation system. The user simply specifies the desired chart, and the system automatically generates it based on the stored data.

For this project, I used the following technologies:

  • Symfony: I handled the backend, creating a RESTful API that connects with the frontend and manages all the logic and database interactions.
  • Angular: The entire frontend was developed in Angular. I used libraries such as chart.js, zxing/ngx-scanner for barcode scanning, OpenLayers for interactive maps, among others. I also integrated the DeepSeek API for AI functionalities.
  • HTML/CSS: Used to structure and style the interface components.
  • Bootstrap: Allowed me to quickly create a well-structured, responsive website.
  • AWS: I utilized AWS services to deploy and host the application in production.
  • MySQL: Used MySQL to create the database.

Alicia Estética -- Sale of products and reservations

This was a group project we carried out for an aesthetic clinic. The main focus was to create a well-crafted and visually appealing frontend, as this was the client’s top priority.

The website features a store section where users can purchase various available products. It also includes a booking section where you can select the service you want with the employee of your choice. Additionally, there are several informational pages explaining everything the clinic offers.

The technologies used in this project were the following:

  • Symfony: Used to develop the backend, creating a RESTful API that connects to the database.
  • Angular: Responsible for the frontend of the application. In this case, we did not use any external libraries.
  • HTML/CSS: Used to structure and style the entire interface.
  • Bootstrap: Essential for making the website fully responsive, which was important since many users access it via mobile devices.
  • Docker: Used to run and contain the application during development.
  • Git: A crucial tool for team collaboration. Each member worked on their own branch and we integrated changes into main.
  • AWS: Used to deploy the application in the cloud.
  • MySQL: Used to manage all the data related to products, clients, and bookings.

EcoBuddy -- Sustainable social network

EcoBuddy was a team project focused on developing a social network aimed at promoting environmentally friendly actions. Users could share activities such as garbage collection, tree planting, and other sustainable initiatives.

The platform featured a points system, where users earned points after completing each activity based on its duration and difficulty. Additionally, trophies were implemented as a gamification method to encourage ongoing user participation.

The technologies used in this project were:

  • PHP: The entire project was developed using pure PHP for both backend and frontend, without any frameworks.
  • HTML/CSS: Used to structure the page and apply custom styling.
  • Git: Each team member worked on their own branch, and changes were merged into the main branch.
  • Docker: Used to set up the development environment and run the application in an isolated container.
  • MySQL: The database was created using MySQL to store all necessary data.
